# 运行最小 Demo

## 启动
在项目根目录执行：

```bash
swift run PetAIDesktop
```

启动后会出现一个透明覆盖层窗口，屏幕上会看到一个 `CAT` 占位体做随机运动。

当前完整 Demo 支持：
- 自动行为：待机、走路、跑步、跳跃（动态地面）
- 点击猫咪：触发原地跳跃
- 拖拽猫咪：把猫咪放到新高度，新高度即新的“地面”
- 右上角控制面板：节奏档位（慵懒/日常/活跃）与暂停开关

## 停止
- 在应用窗口中按 `Escape` 键快速退出
- 或在终端按 `Ctrl + C`
- 或按 `Cmd + Q` （标准 Mac 应用退出）

## 常见问题
1. 看不到占位体
- 检查是否在多显示器副屏；可先只保留主屏测试。
- 检查是否被系统缩放或窗口层级影响。

2. 无法启动
- 先执行：`swift build`
- 再执行：`swift run PetAIDesktop`

3. 控制面板不见了
- 控制面板默认在屏幕右上角。
- 若多显示器场景，请先在主屏测试。

4. 权限问题
- 当前最小 Demo 仅使用透明层渲染，不依赖云服务。
- 后续接入窗口监听时可能需要系统辅助功能授权。
